Typography

Sizing & utility styles.

Use the base scale for readable UI text, then reserve the all-caps eyebrow for orientation, section labels, and metadata.

--font-size-sm

Small supporting text for metadata and helper copy.

--font-size-md

Medium text is the default body size across forms, cards, and system copy.

--font-size-lg

Large text is best for section titles, short callouts, and grouped summaries.

.eyebrow

:System label

Uppercase orientation label for sections, metrics, and restrained hero metadata.
